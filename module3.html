Module 3: Customized Design with JavaScript, HTML, & CSS

Learning Goals
  • Acquire basic Javascript skills for presenting randomized stimuli, debugging, and recording responses
  • Describe how HTML, CSS, and JavaScript relate
  • Identify the differences between JavaScript libraries and how they affect coding strategies
  • Compare and contrast the efficiency and coding of different sample JavaScript scripts

Test Yourself:

Assignments:

  • Program a basic task that I give a description of, w/ four trials. Can have a sample version of this, like a task-switching experiment.
  • Program a basic version of whatever task you’re doing, with say four trials. No need for randomization, just build the stimuli matrices, timelines or callbacks, etc.
  • Program an experiment that displays images, program an experiment that displays text, and program a survey-style study.
  • Read through [instructor link/name of code], figure out what each line of code is doing, and comment the script. Check your comments against this other explanation script.
